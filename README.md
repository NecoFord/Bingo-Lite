# Bingo-Lite
Um jogo de Bingo simplificado desenvolvido em linguagem C, executado via terminal. O programa gera números aleatórios, permite troca de cartela durante a partida e registra os vencedores em arquivo.
Para o funcionamento do jogo, é necessário um arquivo cartelas.txt, esse arquivo deve conter as cartelas disponíveis. Cada linha deve conter 5 números separados por vírgula.
